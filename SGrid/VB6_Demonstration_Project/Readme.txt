S-Grid is an all VB grid giving you the control other grids don't, and 
all in 290kb. Create Outlook-style grids, span column text across rows, 
freely set row and column heights, set colours and fonts for each cell, 
add icons to cells, sort and group data... 

This ZIP contains full source code for the OCX, the release version
of the S-Grid OCX and a demonstration project.

Dependencies
============
S-Grid requires SSubTmr.DLL to run. Download it from:
http://vbaccelerator.com/j-index.asp?url=codelib/ssubtmr/ssubtmr.htm

The demonstration project uses the vbAccelerator ImageList as an 
icon source.  Download it from:
http://vbaccelerator.com/j-index.asp?url=codelib/comctl/imglist.htm

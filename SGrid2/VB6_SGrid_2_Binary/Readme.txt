S-Grid 2.0 is an all VB grid giving you the control other grids don't, 
and all in 480kb. Create Outlook-style grids, span column text across 
rows, freely set row and column heights, set colours and fonts for each 
cell, add icons to cells, sort and group data... 

This ZIP contains the release version of the OCX for VB6.

Dependencies
============
S-Grid 2.0 requires SSubTmr6.DLL to run. Download it from:
http://vbaccelerator.com/article.asp?id=17
